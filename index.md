# Privacy Policy – AutoMessenger

Effective Date: January 3, 2026
 
AutoMessenger is a small, personal-use application created for use with a single eBay seller account. The purpose of this application is to send a one-time courtesy message to buyers after a completed purchase, such as a thank-you message or confirmation of order processing or shipment.

## Information Collection and Use
AutoMessenger does not collect, store, retain, or share any personal data. Buyer and order information is accessed temporarily through the official eBay API solely for the purpose of sending the courtesy message.

## Data Storage
No buyer information, messages, order details, or personal data are stored, logged, or retained by the application. All data is discarded immediately after use.

## Data Sharing
AutoMessenger does not sell, rent, share, or disclose any data to third parties.

## Security
All data access is performed through eBay’s secure OAuth authentication and official APIs.

## Scope
This application is not a public service, SaaS platform, or third-party application. It is intended solely for personal use by the account owner.

## Contact
If you have questions regarding this privacy policy, you may contact the application owner via their eBay seller account.
